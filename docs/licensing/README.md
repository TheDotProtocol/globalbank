# Global Dot Bank — Licensing Roadmap

Primary licensing path for a **full licensed bank** (not partner-bank model):

1. **BVI** — Banking licence via Financial Services Commission (FSC)
2. **Labuan** — Labuan bank (DIB) via Labuan FSA
3. **Thailand** — Digital / commercial bank (Bank of Thailand) — expansion
4. **India** — SFB / NBFC (RBI) — expansion
